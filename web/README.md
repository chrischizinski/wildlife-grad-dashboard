# Wildlife Graduate Dashboard - Web Directory

This directory contains the source code for the frontend dashboard.

## Structure

-   **`wildlife_dashboard.html`**: The main entry point for the application.
-   **`assets/`**:
    -   **`css/enhanced-styles.css`**: The premium design system styles.
    -   **`js/dashboard-core.js`**: Main application logic (JSON fetching, KPI/cards/charts rendering, and Leaflet map rendering).
-   **`data/`**: JSON data files generated by the backend scraper.
    -   `dashboard_analytics.json`: Key metrics and trends.
    -   `enhanced_data.json`: Full dataset with geolocation.
    -   `export_data.json`: Job listings.
-   **`_archive/`**: Contains legacy files and unused data that were cleaned up.
    -   `js/`: Old scripts (`analytics_dashboard.js`, etc.)
    -   `data/`: Raw/unused JSON files (`positions.json`, etc.)
    -   Legacy HTML files.

## Running the Dashboard

Due to CORS restrictions on local file access, run a local server:

```bash
python3 -m http.server
# Visit http://localhost:8000/web/wildlife_dashboard.html
```
